# 🌾 OptiMandi – AI-Powered Agricultural Market Intelligence System

OptiMandi is an AI-driven decision support system designed to optimize agricultural mandi (market) operations by transforming unstructured data into structured, actionable insights.

It helps officials make faster and more accurate decisions regarding procurement, storage, and distribution using AI-based analysis.

---

## 🚀 Problem Statement

Agricultural mandi data is often:
- Stored in unstructured formats (PDFs, Excel sheets, manual reports)
- Difficult and slow to analyze
- Not optimized for real-time decision-making

This leads to:
- Crop wastage due to over-supply
- Price spikes due to under-supply
- Inefficient logistics and resource allocation

---

## 💡 Solution

OptiMandi solves this by:
- Allowing users to upload raw mandi data
- Converting unstructured data into structured format
- Using AI (Google Gemini) to analyze data
- Generating reports, insights, and actionable tasks
- Providing a dashboard for decision-making

---

## 🏗️ System Workflow

1. User logs in / registers
2. Uploads mandi data (PDF / Excel / CSV)
3. Backend parses and structures the data
4. Data is sent to Google Gemini AI
5. AI generates:
   - Market reports
   - Demand-supply insights
   - Actionable recommendations
6. Results are stored in MongoDB
7. Dashboard displays insights for user actions

---

## ⚙️ Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **AI Integration:** Google Gemini API  
- **File Processing:** CSV / Excel / PDF parsers  

---

## 📊 Key Features

- AI-powered market analysis
- Structured report generation from raw data
- Demand vs supply gap detection
- Task-based recommendations (procurement, storage, transport)
- Human-in-the-loop decision approval system
- Dashboard for real-time insights
